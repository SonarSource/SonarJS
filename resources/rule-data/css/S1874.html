Why is this an issue?

CSS features are sometimes deprecated to indicate that they are no longer recommended for use. This is typically because a better alternative exists or because the feature is no longer supported by modern browsers.

Using deprecated CSS selectors, at-rules, or property values can lead to inconsistent rendering across browsers and may stop working entirely as browsers remove support. 

/* Noncompliant: deprecated selector */
acronym {}

/* Noncompliant: deprecated at-rule */
@viewport { width: device-width; }

/* Noncompliant: deprecated keyword value */
a { overflow: overlay; }

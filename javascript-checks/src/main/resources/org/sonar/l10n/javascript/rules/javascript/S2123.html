A value that is incremented or decremented and then not stored is at best wasted code and at worst a bug.

Noncompliant Code Example

function pickNumber() {
  let i = 0;
  i = i++; // Noncompliant; i is still zero

  return i++; // Noncompliant; 0 returned
}

Compliant Solution

function pickNumber() {
  let i = 0;
  i++;

  return ++i;
}
